
# 🏆 复制销冠 - AI智能销售助手

基于 LangChain4j + Spring Boot + Electron 的智能销售辅助系统。

## 项目简介

这是一个基于 LangChain4j 和 Spring Boot 的 AI 智能销售助手系统,通过 RAG 模式实现销售知识库的智能检索和应用。

## 功能特性

- 💼 **面销策略分析**：AI 自动生成销售策略
- 🎙️ **录音智能复盘**：上传录音，AI 分析对话质量
- 📚 **知识库管理**：销售技巧、话术、案例管理
- 🤖 **AI智能问答**：基于 RAG 的智能问答系统

## 核心功能

### 1. 面销前策略分析
- 根据客户信息、产品信息自动生成销售策略
- 提供关键话术要点和可能的异议处理方案
- 推荐最佳沟通方式和成功关键因素

### 2. 会话录音管理
- 上传销售面谈录音文件
- 语音转文字（模拟实现，可集成真实语音API）
- AI 自动分析对话质量
- 生成分数评估和改进建议

### 3. 知识库管理
- 添加销售技巧、话术、案例到知识库
- 支持分类管理和标签检索
- 基于 RAG 的智能检索增强

### 4. AI 智能问答
- 基于知识库的智能问答
- 实时销售指导和建议

## 技术栈

- **后端**: Spring Boot 4.0.6
- **AI框架**: LangChain4j 0.36.0
- **数据库**: H2 (内存数据库)
- **前端**: 原生 HTML/CSS/JavaScript
- **桌面应用**: Electron
- **构建工具**: Maven
